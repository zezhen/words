Find all divisors of a natural number

2017-01-09 21:02:57
Tech

Given a natural number n, print all distinct divisors of it.

Examples:

A Naive Solution can be to iterate all the numbers from 1 to n, checking if that number divides n and printing it.

How to improve above solution? As we can see, (i, n/i) can be a pair if i is the divisor of n, thus we don’t have to check all numbers from 1 to n. Therefore, the question will be where to be stop? The answer is sqrt(n).

Use list is to make divisor in order.
